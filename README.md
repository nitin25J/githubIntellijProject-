# 🎮 Java Mini Projects Collection

This repository contains **three beginner-friendly Java projects** that demonstrate core programming concepts like loops, conditionals, user input, randomness, and GUI development.

---

## 📌 Projects Included

### 1️⃣ Number Guessing Game 🎯
A simple console-based game where the user tries to guess a randomly generated number.

#### 🔹 Features
- Random number generation (1–10)
- User input using Scanner
- Hint system (Too high / Too low)
- Tracks number of attempts

#### 🔹 How It Works
- The program generates a number between 1 and 10
- The user keeps guessing until the correct number is found
- Feedback is given after each attempt

#### 🔹 Concepts Used
- Random class
- Scanner class
- do-while loop
- Conditional statements

---

### 2️⃣ Rock Paper Scissors Game ✊📄✂️
A console-based game where the user plays against the computer.

#### 🔹 Features
- User vs Computer gameplay
- Random computer moves
- Input validation
- Option to play multiple rounds

#### 🔹 How It Works
- User enters rock, paper, or scissors
- Computer randomly selects a move
- Winner is decided based on rules
- User can choose to play again

#### 🔹 Concepts Used
- Arrays
- Random class
- Loops (do-while)
- String handling
- Conditional logic

---

### 3️⃣ Login System (GUI-Based) 🔐
A simple login system built using Java Swing.

#### 🔹 Features
- GUI-based login form
- Predefined user credentials
- Login validation
- Reset button functionality
- Welcome screen after successful login

#### 🔹 How It Works
- User enters ID and password
- System checks credentials using HashMap
- Displays success or error message
- Opens a welcome window on successful login

#### 🔹 Concepts Used
- Java Swing (JFrame, JButton, JTextField, etc.)
- Event Handling (ActionListener)
- HashMap for storing credentials
- GUI layout and design
